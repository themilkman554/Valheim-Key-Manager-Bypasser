# Valheim Key Manager Bypasser

## Why It Exists

This project is a tool designed to bypass the Key Manager in some Valheim mods by patching the `.DLL`, allowing users to run the mods on Linux servers as well as access certain features or content that might otherwise be restricted by scummy paywalls.

## GitHub Pages Link

For more information and access to the project, please visit the [GitHub Pages link](https://themilkman554.github.io/Valheim-Key-Manager-Bypasser/)

## Usage

Simply use the link above, or download the source and open `index.html` in your browser to use the tool. No additional setup or installation is required.
